Quiz Game (Console Based)::

This is a simple **JavaScript Quiz Game** that runs in the browser console.  
The HTML file loads the quiz logic from an external JavaScript file (`quiz.js`).

##  Project Structure:

/quiz-game │── index.html │── quiz.js │── README.md

##  How the Quiz Works?

- The quiz questions are stored in `quiz.js`
- When the page loads, the quiz runs automatically
- Questions and answers appear in the **browser console**
- User inputs answers via `prompt()`
- Score is calculated and displayed at the end

##  How to Run the Project?

1. Download or clone the project
2. Open `index.html` in any modern browser (Chrome recommended)
3. Right-click on the page and select **Inspect**
4. Open the **Console** tab
5. Follow the quiz instructions shown in the console

##  Technologies Used

- HTML5
- JavaScript (Vanilla JS)

##  Note:

This quiz is **console-based**, so no input/output appears on the webpage itself.  
All interactions happen inside the browser console.

:: How to Run the Quiz ?

1. Create an index.html file and link the JavaScript file:

5. View your score at the end.


6. Choose whether to replay or exit.
